# 1. Ders - Unity Nedir ve Kurulumu Adımları

## Unity Nedir?

Unity, 2D ve 3D oyunlar, simülasyonlar ve interaktif deneyimler geliştirmek için kullanılan güçlü bir oyun motorudur.

## Kurulum Adımları

### 1. Unity Hub İndirme
- [Unity Hub](https://unity.com/download) sayfasından Unity Hub'ı indirin
- Unity Hub, farklı Unity sürümlerini yönetmenizi sağlar

### 2. Unity Hesabı Oluşturma
- Unity ID oluşturun (ücretsiz)
- Lisans aktivasyonu için gerekli

### 3. Lisans Seçimi
- **Personal**: Ücretsiz, yıllık geliri $100K altında olanlar için
- **Plus**: Küçük stüdyolar için
- **Pro**: Profesyonel kullanım için

### 4. Unity Editör Kurulumu
- Unity Hub üzerinden istediğiniz sürümü kurun
- LTS (Long Term Support) sürümleri önerilir

## Notlar

- Kurulum sırasında gerekli modülleri seçmeyi unutmayın (Android, iOS, WebGL vb.)
- Visual Studio veya VS Code entegrasyonu için gerekli bileşenleri kurun
